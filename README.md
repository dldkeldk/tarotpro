## ✨ 주요 기능

- **실시간 상담 답변 보조**: 카드의 상징과 내담자의 상황을 결합하여 상담사가 바로 활용할 수 있는 답변 초안 제안.
- **상담 흐름 및 감정 분석**: 대화의 핵심 키워드를 추출하고 운의 흐름을 대시보드에 시각화.
- **체계적인 세션 로그**: 모든 상담 내역을 자동으로 요약 및 저장하여 상담사의 자산으로 관리.
- **프리미엄 UI/UX**: 상담의 본질에 집중할 수 있도록 설계된 세련된 글래스모피즘 디자인.

## 🚀 시작하기

### 사전 준비 사항
- **Node.js** (최신 안정 버전 권장)
- **Google Gemini API Key**

### 설치 및 실행
1. 저장소 클론:
   ```bash
   git clone [repository-url]
   ```
2. 의존성 설치:
   ```bash
   npm install
   ```
3. 환경 변수 설정:
   - `.env.example` 파일을 복사하여 `.env` 파일을 생성하고 `GEMINI_API_KEY`를 입력하세요.
4. 로컬 서버 실행:
   ```bash
   npm run dev
   ```

## 🛠 기술 스택
- **Frontend**: React 19, TypeScript, Tailwind CSS (v4), Motion (Framer Motion)
- **Backend**: Express (Vite Dev Middleware integration)
- **AI Engine**: Google Generative AI (Gemini)
